# 회의록 → 실행 항목 변환기

회의 녹취나 메모를 붙여넣으면 AI가 자동으로 정리해주는 웹 앱입니다.

## 변환 항목

- **결정 사항** — 회의에서 확정된 내용 (중요도 표시)
- **미결정 사항** — 아직 결론 나지 않은 건 (사유 포함)
- **담당자별 할 일** — 누가 무엇을 언제까지
- **일정 리스크** — 지연/위험 요소 (심각도 표시)
- **후속 질문** — 추가 확인이 필요한 사항
- **공지/메일 초안** — 바로 발송 가능한 요약문

## 시작하기

```bash
# 의존성 설치
npm install

# 환경변수 설정
# .env.local 파일에 ANTHROPIC_API_KEY 입력

# 개발 서버 실행
npm run dev
```

http://localhost:3000 에서 확인하세요.

## 기술 스택

- Next.js 16 + TypeScript
- Tailwind CSS 4
- Claude API (Anthropic SDK)
